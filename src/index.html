Про меня

Меня зовут Дмитрий

Имею опыт разработки web-приложений в учебных целях более года.
Ранее работал в продажах и решил сменить сферу деятельности с менеджера по продажам на разработчика, потому что всегда была интересна IT-сфера, но считал что программирование это очень сложно и скучно, прочитав свою первую книгу по программированию, убедился в обратном.
За время своего обучения изучил основы HTML, CSS, JavaScript и React, в результате создал несколько сайтов, и разместил их в блоке с портфолио, так же имею учебный опыт разработки на языке Python с помощью которого создал несколько backend-приложений используя фреймворк Django. Планирую дальше развиваться в frontend разработке потому что это направление мне наиболее интересно.

Frontend-разработка
Адаптивная верстка, знание препроцессоров SASS, SCSS, разработка web-приложений используя нативный JavaScript или фреймворк React.
Backend-разработка
Написание не сложных backend приложений, с помощью фреймворка Django, работа с реляционными базами данных (Postgree, MySQL).

Навыки

Стэк технологий которым я владею
HTML5
Именно он создает каркас вашего сайта или приложения, а пятая версия позволит мне создавать более SEO-оптимизированную структуру вашего продукта
CSS3
Этот язык стилей позволяет мне создавать абсолютно любой внешний вид вашего сайта или приложения. Все ограничивается только вашей фантазией!
Java Script
Этот язык программирования позволяет оживить все что угодно: слайдеры, окна, подсказки, вкладки, получение данных от сервера и многое другое
React
Эта библиотека позволяет создавать web-приложения. Мы можем создать максимально интерактивный продукт именно под ваши цели
Redux
Redux — библиотека для JavaScript с открытым исходным кодом, предназначенная для управления состоянием приложения.
Python
Python — это высокоуровневый язык программирования, который используется в различных сферах IT, таких как машинное обучение, разработка приложений, web, парсинг и другие.
Django
Фреймворк для веб-приложений на языке Python, использующий шаблон проектирования MVC
MySQL
MySQL — это система управления реляционными базами данных (СУРБД). Где «реляционные» значит, что данные хранятся в виде таблиц.
